## Tailwind-styled Button 

React button for illustrating tailwind styling, supported via the component's env's tailwind configuration

### Component usage
```js
<TwButton text="Styled By Tailwind" className="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded p-20"/>
```

### Play with tailwind styling

Add tailwind styles and see the button change:
```js live
<TwButton text="Styled By Tailwind" className="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded p-20"/>
```
